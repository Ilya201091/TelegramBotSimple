
<h1>Telegram Bot для магазина</h1>
MyBotMeatDay

<h2>Функционал</h2>
<ul>
 <li>Запрос имени</li>
 <li>Контактный телефон</li>
 <li>Заказ</li>
 <li>Проинформировать, что заказ будет собран</li>
 <li>Оповещение, что заказ готов и общая стоимость</li>
</ul>
